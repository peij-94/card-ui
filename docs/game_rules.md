# 📜 元素之战 - 游戏规则文档

## 游戏概述

**游戏名称**: 元素之战 (Elemental Battle)
**类型**: 回合制卡牌对战
**人数**: 1v1 或 2v2
**游戏时长**: 10-20分钟

---

## 核心机制

### 1. 胜利条件
- **标准模式**: 将对手生命值降至0
- **特殊模式**: 完成特定任务（如先召唤3个传说单位）

### 2. 基础属性

#### 玩家属性
| 属性 | 初始值 | 上限 | 说明 |
|------|--------|------|------|
| 生命值 | 100 | 100 | 降至0时失败 |
| 法力值 | 0 | 10 | 每回合自动恢复 |
| 手牌上限 | 10 | 10 | 超出时弃牌 |
| 牌库 | 30 | - | 抽完时受到疲劳伤害 |

#### 战场位置
```
┌─────────────────────────────────────┐
│         敌方后排 (3格)               │
│         敌方前排 (3格)               │
├─────────────────────────────────────┤
│         我方前排 (3格)               │
│         我方后排 (3格)               │
└─────────────────────────────────────┘
```

## 回合流程

### 回合开始阶段
1. **法力恢复**: 恢复所有法力值
2. **抽牌**: 从牌库抽取2张卡牌
3. **触发效果**: 触发"回合开始"类效果

### 主要阶段（可任意顺序执行）
1. **打出卡牌**: 消耗法力打出卡牌
2. **移动单位**: 移动战场上的召唤单位
3. **使用技能**: 激活单位技能
4. **攻击**: 命令单位攻击或直接使用攻击卡牌

### 回合结束阶段
1. **触发效果**: 触发"回合结束"类效果
2. **弃牌**: 手牌超过上限时弃至10张
3. **回合切换**: 轮到对手回合

## 卡牌类型详解

### 1. 攻击卡 (Attack)
- **效果**: 直接对目标造成伤害
- **特点**: 即时生效，不留在战场
- **目标**: 敌方单位或玩家

**示例**: 
- 烈焰斩击: 造成15点火焰伤害
- 雷霆一击: 造成20点雷电伤害，30%麻痹

### 2. 防御卡 (Defense)
- **效果**: 获得护盾或减伤
- **特点**: 即时生效，部分持续多回合
- **目标**: 我方玩家或单位

**示例**:
- 寒冰壁垒: 获得20点护盾，持续3回合
- 山岳之盾: 获得35点护盾，护盾存在时攻击力+10%

### 3. 法术卡 (Spell)
- **效果**: 各种特殊效果
- **特点**: 即时生效，效果多样
- **目标**: 根据法术而定

**示例**:
- 暴风雪: 对所有敌人造成18点冰霜伤害
- 治愈之泉: 为所有友方回复15生命

### 4. 召唤卡 (Summon)
- **效果**: 召唤单位到战场
- **特点**: 单位留在战场，可攻击/被攻击
- **目标**: 我方空位

**示例**:
- 炼狱战车: 召唤攻击力35，生命值80的单位
- 天使降临: 召唤攻击力35，每回合为友方回复15生命的单位

## 属性相克系统

### 相克关系
```
火 → 风 → 雷 → 水 → 火
土 → 雷 → 冰 → 自然 → 土
光 ↔ 暗影 (互相克制)
```

### 克制效果
- **优势属性**: 伤害 +30%
- **劣势属性**: 伤害 -20%
- **相同属性**: 伤害不变

### 特殊交互
| 属性组合 | 效果 |
|----------|------|
| 火 + 风 | 火焰伤害范围扩大 |
| 水 + 冰 | 冰冻概率提高 |
| 雷 + 水 | 连锁闪电目标+1 |
| 光 + 暗影 | 双方伤害+50% |

## 稀有度与平衡

### 稀有度分布
| 稀有度 | 单卡组最大数量 | 特点 |
|--------|----------------|------|
| 普通 | 无限制 | 稳定可靠 |
| 稀有 | 最多6张 | 有特效 |
| 史诗 | 最多4张 | 强力效果 |
| 传说 | 最多2张 | 改变战局 |

### 法力曲线建议
```
1费: 4-6张 (前期节奏)
2费: 6-8张 (主要战力)
3费: 6-8张 (核心卡牌)
4费: 4-6张 (中期发力)
5费+: 2-4张 (终结手段)
```

## 连击系统

### 连击触发
连续打出同属性卡牌触发连击

### 连击奖励
| 连击数 | 奖励 |
|--------|------|
| 2连击 | 第2张卡牌伤害+20% |
| 3连击 | 第3张卡牌伤害+50% |
| 4连击 | 第4张卡牌伤害+100%，抽1张牌 |
| 5连击+ | 所有后续同属性卡牌伤害+100%，每回合抽2张牌 |

### 连击中断
- 打出不同属性卡牌
- 回合结束
- 被敌方控制技能打断

## 进阶机制

### 1. 元素共鸣
场上同时存在3个及以上同属性单位时触发:
- **火之共鸣**: 所有火属性单位攻击力+20%
- **冰之共鸣**: 所有敌方单位速度-20%
- **雷之共鸣**: 所有雷属性攻击有50%连锁
- **其他属性**: 类似效果

### 2. 环境效果
战场可能带有环境效果:
- **火山**: 每回合所有单位受到5点火焰伤害
- **暴风雪**: 每回合30%概率冰冻随机单位
- **雷云**: 每回合随机对1个单位造成10点雷电伤害
- **森林**: 自然属性单位每回合回复5生命

### 3. 终极技能
当法力值满(10点)时可释放:
- 消耗所有法力
- 释放强力的终极技能
- 每个系别有独特的终极技能

## 游戏模式

### 1. 标准对战
- 30张牌组
- 100生命值
- 无特殊规则

### 2. 快速模式
- 20张牌组
- 50生命值
- 初始法力+2

### 3. 无限模式
- 牌库无限
- 每回合随机获得卡牌
- 看谁能坚持更多回合

### 4. 冒险模式 (PVE)
- 挑战AI控制的Boss
- Boss有特殊技能和阶段
- 通关获得奖励卡牌

## 卡组构建建议

### 进攻型卡组
- 攻击卡: 40%
- 法术卡: 30%
- 召唤卡: 20%
- 防御卡: 10%
- **策略**: 快速压低对手血量

### 控制型卡组
- 法术卡: 40%
- 防御卡: 30%
- 召唤卡: 20%
- 攻击卡: 10%
- **策略**: 控制场面，消耗对手资源

### 召唤型卡组
- 召唤卡: 50%
- 法术卡: 20%
- 防御卡: 20%
- 攻击卡: 10%
- **策略**: 用数量优势压制对手

### 平衡型卡组
- 平均分配各类型卡牌
- **策略**: 灵活应对各种情况

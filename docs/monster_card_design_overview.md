## 第一章：齿轮连携系统核心机制

### 1.1 系统概述

**齿轮连携系统**是一套深度策略卡牌战斗机制，专为蒸汽朋克主题设计。核心特点是"齿轮蓄能与连锁反应"，让每张卡牌的出牌都像精密齿轮一样环环相扣。

### 1.2 核心资源：齿轮

#### 齿轮获取
| 来源 | 齿轮数量 | 说明 |
|------|---------|------|
| 每回合基础 | 3点 | 自动获得 |
| 击杀怪物 | +1点 | 战斗奖励 |
| 蓄力攻击 | +1点 | 特殊攻击 |
| 被动技能 | +1点 | 部分怪物特有 |

#### 齿轮消耗
- 普通卡牌：1-2点
- 稀有卡牌：3-4点
- 史诗卡牌：5-6点
- 传说/终极卡牌：7-10点

### 1.3 连锁系统

#### 连锁类型

**1. 基础连锁**
- 条件：连续使用3张卡牌
- 效果：第4张卡牌伤害+30%

**2. 齿轮连锁**
- 条件：本回合齿轮费用总和达到阈值
- 效果：触发额外效果或追加伤害

**3. 元素连锁**
- 条件：连续使用相同元素卡牌
- 效果：触发元素共鸣，造成额外元素伤害

**4. 终极连锁（Boss专属）**
- 条件：满足特定条件（如生命值低于30%）
- 效果：释放专属终极技能

#### 连锁伤害公式
```
最终伤害 = 基础伤害 × (1 + 连锁加成) × (1 + 元素加成) × (1 + 特殊加成)
```

### 1.4 过热系统

#### 过热状态
- 当单回合齿轮消耗超过5点时，触发过热
- 过热效果：下回合齿轮获取-1

#### 过热爆发
- 消耗全部齿轮（≥5）
- 造成大量伤害
- 特殊：可与连锁系统叠加

### 1.7 战斗流程

```
┌─────────────────────────────────────┐
│            战斗开始                  │
├─────────────────────────────────────┤
│  1. 初始回合                        │
│     - 抽3张牌                       │
│     - 获得3齿轮                     │
│     - 触发被动技能（如有）           │
├─────────────────────────────────────┤
│  2. 行动阶段                        │
│     - 打出卡牌                      │
│     - 计算伤害                      │
│     - 触发连锁                      │
│     - 应用效果                      │
│     - （重复直到本回合结束）          │
├─────────────────────────────────────┤
│  3. 回合结束                        │
│     - 状态效果倒计时                │
│     - 持续伤害结算                  │
│     - 检查连锁条件                  │
│     - 弃牌（手牌上限7张）           │
├─────────────────────────────────────┤
│  4. 敌人回合                        │
│     - AI出牌                       │
│     - 结算敌人行动                  │
├─────────────────────────────────────┤
│  5. 检查战斗结果                     │
│     - HP≤0 → 胜利                  │
│     - HP>0 → 下一回合               │
└─────────────────────────────────────┘
```

### 1.8 连锁组合示例

#### 火焰怪物连锁
```
第1张：火焰喷射（2齿轮）→ 基础伤害
第2张：燃烧突击（2齿轮）→ 火焰连锁激活
第3张：过热爆发（3齿轮）→ 连锁加成+30%
第4张：熔岩冲击（4齿轮）→ 终极连锁，伤害爆炸
```

#### 雷电怪物连锁
```
第1张：闪电箭（2齿轮）→ 基础伤害
第2张：静电冲击（2齿轮）→ 雷电连锁激活
第3张：连锁闪电（3齿轮）→ 闪电跳跃至3目标
第4张：雷霆万钧（4齿轮）→ 终极连锁+麻痹
```

### 1.9 卡牌稀有度

| 稀有度 | 符号 | 齿轮消耗 | 效果强度 | 获取概率 |
|-------|------|---------|---------|---------|
| 普通 | ⭐ | 1-2 | 基准 | 60% |
| 稀有 | ⭐⭐ | 2-3 | 1.2× | 25% |
| 史诗 | ⭐⭐⭐ | 3-4 | 1.5× | 10% |
| 传说 | ⭐⭐⭐⭐ | 4-5 | 2.0× | 4% |
| 终极 | ⭐⭐⭐⭐⭐ | 6+ | 2.5×+ | 1% |

### 1.10 设计平衡原则

1. **能量守恒**：高伤害卡牌 = 高齿轮消耗
2. **策略多样性**：至少3种通关方式
3. **反制机制**：每种强卡都有反制手段
4. **随机性适度**：运气影响30%，技术影响70%
5. **成长曲线**：越后期越需要策略

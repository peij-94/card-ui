# 🎬 卡牌动画效果设计文档

## 动画风格
- **风格**: 像素风动画，流畅但保留复古感
- **帧率**: 12-24 FPS
- **缓动函数**: 主要使用 ease-out 和 ease-in-out
- **时长**: 快速反馈 0.2-0.3s，主要动画 0.5-0.8s

---

## 基础动画

### 1. 抽牌动画
**触发**: 从牌库抽取卡牌到手牌

```
动画序列:
1. 卡牌从牌库位置弹出 (0.1s, scale 0.8→1.0)
2. 卡牌翻转显示正面 (0.3s, rotateY 0°→90°→0°)
3. 卡牌飞到手牌区域 (0.4s, ease-out)
4. 手牌重新排列 (0.2s, 所有手牌调整位置)

特效:
- 飞行轨迹留下淡淡的光痕
- 稀有度越高，光痕颜色越明显
- 传说卡牌抽牌时有全屏闪光效果
```

### 2. 打出动画
**触发**: 将手牌打出到战场

```
动画序列:
1. 卡牌从手牌区域抬起 (0.1s, scale 1.0→1.1, y-20px)
2. 卡牌飞向目标位置 (0.3s, ease-in-out)
3. 卡牌落下并轻微弹跳 (0.2s, ease-out)
4. 触发卡牌特效动画

特效:
- 飞行时卡牌边缘发光
- 落地时产生冲击波（根据稀有度）
- 传说卡牌打出时有慢动作效果
```

### 3. 攻击动画
**触发**: 攻击型卡牌发动攻击

```
动画序列:
1. 攻击卡牌向前冲刺 (0.15s, x+50px)
2. 攻击命中目标 (0.1s)
3. 目标受击抖动 (0.2s, 左右抖动3次)
4. 攻击卡牌返回原位 (0.2s)

特效:
- 冲刺时留下残影
- 命中时产生命中特效（根据属性）
  - 火系: 爆炸火焰
  - 冰系: 冰霜碎裂
  - 雷系: 闪电
  - 等等...
- 目标显示伤害数字飘字
```

### 4. 防御动画
**触发**: 防御型卡牌激活护盾

```
动画序列:
1. 防御卡牌旋转发光 (0.3s, rotate 0°→360°)
2. 护盾从卡牌扩散 (0.4s, scale 0→1.5→1.0)
3. 护盾持续闪烁 (循环)

特效:
- 护盾为半透明圆顶
- 颜色对应属性系别
- 受到攻击时护盾产生涟漪效果
```

### 5. 法术动画
**触发**: 法术卡牌释放技能

```
动画序列:
1. 法术卡牌悬浮并旋转 (0.5s, y-30px, rotate 0°→720°)
2. 法术能量聚集 (0.3s, 中心发光)
3. 法术释放 (0.4s)
4. 卡牌回到原位

特效:
- 根据法术类型显示不同特效
- 全屏或局部特效
- 目标区域预警提示
```

### 6. 召唤动画
**触发**: 召唤卡牌召唤单位

```
动画序列:
1. 召唤卡牌打开传送门 (0.4s, 中心漩涡效果)
2. 召唤单位从传送门出现 (0.5s, scale 0→1.0)
3. 召唤单位落地 (0.2s, 轻微弹跳)
4. 传送门关闭

特效:
- 传送门颜色对应属性
- 召唤单位出现时带有属性粒子
- 传说召唤有额外登场特效
```

## 特殊动画

### 7. 卡牌翻转
**触发**: 卡牌从背面翻到正面/反面

```
动画序列:
1. 卡牌缩小并旋转90° (0.15s)
2. 切换正反面贴图 (瞬间)
3. 卡牌恢复正常并旋转90° (0.15s)

缓动: ease-in-out
```

### 8. 卡牌销毁
**触发**: 卡牌被消耗或破坏

```
动画序列:
1. 卡牌闪烁 (0.2s, 透明度变化3次)
2. 卡牌碎裂成像素块 (0.3s)
3. 像素块消散 (0.4s)

特效:
- 碎裂方向根据攻击方向
- 消散时有粒子效果
- 传说卡牌有特殊消散特效
```

### 9. 卡牌升级
**触发**: 卡牌稀有度提升

```
动画序列:
1. 卡牌被光芒包围 (0.3s)
2. 光芒爆发 (0.2s, 全屏闪光)
3. 卡牌显示新边框 (0.3s, 边框颜色渐变)
4. 新特效环绕卡牌 (0.5s)

特效:
- 普通→稀有: 蓝色光芒
- 稀有→史诗: 紫色光芒
- 史诗→传说: 金色光芒
```

### 10. 连击动画
**触发**: 连续打出多张卡牌

```
动画序列:
1. 第一张卡牌正常打出
2. 第二张卡牌加速打出 (0.7倍时间)
3. 第三张卡牌更快打出 (0.5倍时间)
4. 连击数字显示 (在屏幕中央)
5. 连击结束时的终结特效

特效:
- 连击数字逐渐变大
- 2连击: 小特效
- 3连击: 中特效
- 4连击以上: 大特效 + 全屏震动
```

## 技术实现建议

### CSS动画类名规范
```css
.card-draw        /* 抽牌 */
.card-play        /* 打出 */
.card-attack      /* 攻击 */
.card-defend      /* 防御 */
.card-spell       /* 法术 */
.card-summon      /* 召唤 */
.card-flip        /* 翻转 */
.card-destroy     /* 销毁 */
.card-upgrade     /* 升级 */
.combo-x2         /* 2连击 */
.combo-x3         /* 3连击 */
.combo-x4-plus    /* 4连击以上 */
```

### 动画时长常量
```javascript
const ANIMATION_DURATION = {
  FAST: 200,      // 快速反馈
  NORMAL: 500,    // 标准动画
  SLOW: 800,      // 重要动画
  EPIC: 1200      // 传说特效
};
```

### 缓动函数
```javascript
const EASING = {
  easeOut: 'cubic-bezier(0, 0, 0.2, 1)',
  easeInOut: 'cubic-bezier(0.4, 0, 0.2, 1)',
  bounce: 'cubic-bezier(0.68, -0.55, 0.265, 1.55)',
  elastic: 'cubic-bezier(0.175, 0.885, 0.32, 1.275)'
};
```

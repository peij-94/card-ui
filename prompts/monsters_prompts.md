# 🦾 蒸汽朋克怪物设计文档

## 整体风格
- **风格**: 蒸汽朋克 + 奇幻魔法
- **色调**: 金属铜/铁/金色 + 魔法光效
- **特征**: 齿轮、管道、蒸汽、能量核心
- **背景**: 纯黑色

## 生成参数

### 尺寸
- 普通怪: 256x256 或 512x512
- 精英怪: 512x512
- Boss: 1024x1024 (square_hd)

### 风格统一
- 所有怪物: 8-bit retro game style
- 背景: 纯黑色
- 视角: 正面或45度侧面
- 光影: 顶部光源，底部阴影

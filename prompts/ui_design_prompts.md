# 🖥️ 游戏界面UI设计提示词

## 整体风格
- 风格：像素风 + 魔幻元素
- 色调：深色背景 + 属性色系点缀
- 分辨率：1920x1080 (16:9)

---

## 1. 主界面 (Main Menu)

Pixel art game main menu UI, dark fantasy theme, black background with subtle magical rune patterns, title "元素之战" in golden pixel font at top center, menu buttons with pixel borders:
- 开始游戏 (Start Game) - glowing blue
- 卡组编辑 (Deck Editor) - glowing green  
- 冒险模式 (Adventure) - glowing purple
- 设置 (Settings) - glowing gray
- 退出 (Exit) - glowing red
Decorative pixel art elements: floating elemental orbs (fire, ice, lightning), 8-bit retro game style, clean layout, mystical atmosphere

---

## 2. 对战界面 (Battle Screen)

### 完整对战场景
Pixel art card battle game interface, 1920x1080 resolution, dark wooden table background, divided into zones:

**Top Area (Enemy)**:
- Enemy avatar (left) with health bar (red)
- Enemy mana crystals (right) - blue glowing orbs
- Enemy name and rank
- Enemy hand cards (backs only, fanned)

**Middle Area (Battlefield)**:
- 3x2 grid for unit placement
- Each slot has subtle glow when occupied
- Health bars above units
- Attribute icons below units

**Bottom Area (Player)**:
- Player hand cards (5-7 cards fanned at bottom center)
- Player avatar (right) with health bar (green)
- Player mana crystals (left) - blue glowing orbs
- End turn button (bottom right) - golden pixel button

**Side Panels**:
- Deck pile (left) - card back stack
- Graveyard (right) - dark portal icon
- Card detail preview (when hovering) - large card display

Decorative elements: Elemental energy flowing between zones, pixel particle effects, 8-bit retro game style

---

## 3. 手牌区域 (Hand Area)

Pixel art card hand holder, semicircular arrangement at bottom of screen, dark wooden texture base, 5-7 card slots with slight overlap, each card tilted at different angle (fan shape), glowing edge on playable cards (green glow when enough mana, red when not), card backs visible for enemy hand at top, 8-bit retro game style

---

## 4. 战场格子 (Battlefield Grid)

Pixel art battle grid, 3 columns x 2 rows, dark stone tile texture, each cell has:
- Subtle border glow when empty
- Bright glow when occupied
- Small health bar at top of cell
- Attribute symbol at bottom of cell
Grid lines are golden pixel lines, corners have decorative pixel patterns, 8-bit retro game style

---

## 5. 玩家信息面板 (Player Panel)

Pixel art player info UI panel, vertical layout on right side, dark translucent background with golden border:
- Player avatar (circular, pixel art portrait)
- Player name (pixel font)
- Health bar (green gradient, pixel style)
- Mana crystals (10 blue orbs in 2 rows)
- Deck count (number with card icon)
- Graveyard count (number with skull icon)
- Buff/Debuff icons (small pixel icons in row)
8-bit retro game style, clean readable layout

---

## 6. 卡牌详情面板 (Card Detail)

Pixel art card detail view, large card display on right side when hovering, dark translucent panel with golden border:
- Large card image (2x size)
- Card name (pixel font, colored by rarity)
- Attribute icon
- Type icon (attack/defense/spell/summon)
- Cost number (large, blue crystal icon)
- Attack/Defense stats (if applicable)
- Description text (pixel font, wrapped)
- Effect text (pixel font, golden color)
- Rarity indicator (border color and gem)
8-bit retro game style

---

## 7. 回合指示器 (Turn Indicator)

Pixel art turn indicator, top center of screen, banner style with golden pixel border:
- "我方回合" (Your Turn) - blue glowing
- "敌方回合" (Enemy Turn) - red glowing
- Timer (sandglass pixel icon with countdown)
- Turn number counter
Animated glowing effect, 8-bit retro game style

---

## 8. 伤害数字 (Damage Numbers)

Pixel art damage number popup, floating text animation:
- Normal damage: white pixel numbers
- Critical damage: yellow pixel numbers with "!"
- Fire damage: orange-red gradient numbers with flame particles
- Ice damage: cyan-white gradient numbers with snow particles
- Lightning damage: yellow-purple gradient numbers with spark particles
- Healing: green numbers with "+" prefix and heart particles
Bold pixel font, slight bounce animation, fade out after 1 second

---

## 9. 卡组编辑器 (Deck Editor)

Pixel art deck builder interface, split screen layout:

**Left Panel (Collection)**:
- Filter buttons by attribute (9 element icons)
- Filter buttons by rarity (4 rarity gems)
- Filter buttons by type (4 type icons)
- Card grid display (scrollable)
- Search bar (pixel input field)

**Right Panel (Current Deck)**:
- Deck name input (pixel text field)
- Card count (30/30 indicator)
- Mana curve graph (pixel bar chart)
- Deck card list (scrollable, grouped by cost)
- Save/Load buttons

**Center**: Card detail preview when selected

Dark wooden background, golden pixel borders, 8-bit retro game style

---

## 10. 设置界面 (Settings)

Pixel art settings menu, dark background with subtle rune patterns, categories on left:
- 音效 (Sound) - speaker icon
- 音乐 (Music) - music note icon
- 画面 (Graphics) - monitor icon
- 控制 (Controls) - controller icon
- 语言 (Language) - globe icon

Each category has:
- Toggle switches (pixel style on/off)
- Sliders (pixel style with percentage)
- Dropdown menus (pixel style)
- Back button (bottom left)

Golden pixel borders, consistent spacing, 8-bit retro game style

---

## 11. 结算界面 (Victory/Defeat)

Pixel art game result screen, full screen overlay:

**Victory**:
- "胜利!" (VICTORY!) in large golden pixel font with glow
- Rewards display (cards, gold, exp)
- Statistics (damage dealt, turns taken, cards played)
- Continue button (golden pixel button)
- Confetti particle effects (pixel style)

**Defeat**:
- "失败" (DEFEAT) in dark red pixel font
- Tips for improvement
- Retry button
- Back to menu button
- Dark mist particle effects

Background shows blurred battlefield, 8-bit retro game style

---

## 12. 加载界面 (Loading Screen)

Pixel art loading screen, dark background with elemental symbols rotating:
- Central loading icon (spinning elemental orb)
- Loading bar (pixel style, golden fill)
- Loading tips (pixel text, cycling)
- Game logo (top center)
- Version number (bottom right)

Subtle particle effects, smooth animation, 8-bit retro game style

## 设计规范

- **风格**: 16-bit 像素风，复古游戏美学
- **尺寸**: 864 x 1152 (portrait 4:3)
- **背景**: 纯黑底
- **字体**: 小尺寸中文像素字体
- **调色板**: 256色

## 卡牌数据

每张卡牌包含以下属性：
- `id`: 唯一标识符
- `name`: 中文名称
- `attribute`: 属性系别
- `rarity`: 稀有度
- `type`: 卡牌类型 (attack/defense/spell/summon)
- `cost`: 法力消耗 (1-9)
- `attack`: 攻击力
- `defense`: 防御力
- `description`: 卡牌描述
- `effect`: 技能效果

详见 `data/cards_data.json`

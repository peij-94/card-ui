# 蒸汽朋克怪物战斗卡牌游戏

## Steampunk Monster Battle Card Game

一个完整的蒸汽朋克主题卡牌游戏设计，包含28种独特怪物和336张战斗卡牌。

## 🎮 项目概述

本项目包含：

- **28种蒸汽朋克风格怪物**
  - 16种普通怪物
  - 8种精英怪物
  - 4个Boss级怪物

- **336张独特卡牌**
  - 普通怪物卡牌（铜色边框）: 119张
  - 精英怪物卡牌（银色边框）: 68张
  - Boss卡牌（金色边框）: 46张

## ⚙️ 齿轮连携系统

本游戏采用创新的**齿轮连携系统**：

### 核心机制

| 机制 | 说明 |
|------|------|
| **齿轮能量** | 每回合获得3点齿轮能量，用于出牌 |
| **连锁加成** | 连续使用卡牌触发连锁，伤害最高+50% |
| **过热系统** | 高消耗卡牌触发过热，下回合齿轮-1 |
| **元素反应** | 火/冰/雷/水/蒸汽/虚空六种元素 |

### 元素系统

| 元素 | 克制 | 效果 |
|------|------|------|
| 🔥 火 | 克制冰 | 灼烧 |
| ❄️ 冰 | 克制雷 | 冻结 |
| ⚡ 雷 | 克制水 | 麻痹 |
| 💧 水 | 克制火 | 潮湿 |
| ⚙️ 蒸汽 | 无 | 过热 |
| 🌑 虚空 | 无 | 腐蚀 |

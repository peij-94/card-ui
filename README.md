# 🎴 Pixel Battle Card UI

像素风战斗卡牌资产库 — 9大属性系别，共90张卡牌

## 属性系别

| 系别 | 数量 | 主色调 | 边框风格 |
|------|------|--------|----------|
| 🔥 火属性 | 10张 | 红/橙/金 | 金色火焰纹理 |
| 🧊 冰属性 | 10张 | 冰蓝/银 | 银蓝霜冻纹理 |
| 💀 暗影属性 | 10张 | 紫/黑 | 黑曜石符文纹理 |
| ⚡ 雷电属性 | 10张 | 黄/电蓝 | 金色闪电纹理 |
| 🌪️ 风暴属性 | 10张 | 青绿/白 | 银色风纹 |
| 🪨 大地属性 | 10张 | 棕/绿 | 石色岩石纹理 |
| ✨ 圣光属性 | 10张 | 金/白 | 圣洁光芒纹理 |
| 🌿 自然属性 | 10张 | 绿/翠绿 | 藤蔓叶纹 |
| 🌊 水流属性 | 10张 | 深蓝/青 | 波浪纹理 |

## 文件结构

```
├── cards/
│   ├── fire/       # 火属性卡牌
│   ├── ice/        # 冰属性卡牌
│   ├── shadow/     # 暗影属性卡牌
│   ├── thunder/    # 雷电属性卡牌
│   ├── wind/       # 风暴属性卡牌
│   ├── earth/      # 大地属性卡牌
│   ├── holy/       # 圣光属性卡牌
│   ├── nature/     # 自然属性卡牌
│   └── water/      # 水流属性卡牌
└── prompts/        # 卡牌生成提示词
    ├── card_prompts.md      # 火/冰/暗影 提示词
    └── card_prompts_new6.md # 雷电/风暴/大地/圣光/自然/水流 提示词
```

## 设计规范

- **风格**: 16-bit 像素风，复古游戏美学
- **尺寸**: 864 x 1152 (portrait 4:3)
- **背景**: 纯黑底
- **字体**: 小尺寸中文像素字体
- **调色板**: 256色
- **无数据**: 卡牌不显示数值/属性/费用等数据
